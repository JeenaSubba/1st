History Tim Berners-Lee invented = > www in 1989 1990s from 1991 to 1998, HTML developed from version 1 to version 4. World Wide Web Consortium (W3C) recommended XHTML 1.0. WHATWG ( the web Hypertext Application Technology group) was formed in response to slow W3 development, and W3C's decision to close down the development of HTML, in favor of XHTML. In 2006, W3C announced that they would support WHATWG. In 2008, the first HTML5 piblic draft was released. Core Web Technologies

HTML and CSS are two of the core technologies for building web pages.

What is HTML?

HTML stands for HyperText Markup Language. HTML is the standard markup language for describing the structure of web pages.

What is CSS?

CSS stands for Cascading Style Sheet. CSS allows you to specify various style properties for HTML elements such as colors, backgrounds, fonts etc.
